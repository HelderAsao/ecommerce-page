# E-commerce Product Page

Aplicação de página de produto desenvolvida em React que permite visualizar imagens do produto, selecionar quantidade, adicionar itens ao carrinho e visualizar o total da compra.

# Deploy
Acesse o projeto online:
https://ecommerce-page-mauve.vercel.app/

## 🛠 Tecnologias Utilizadas
- React
- JavaScript (ES6+)
- HTML5
- CSS3
- Typescript
- Tailwind CSS

## 🚧 Desafios e Aprendizados
Durante o desenvolvimento deste projeto enfrentei alguns desafios importantes:

- Gerenciar o estado do carrinho e quantidade de itens.
- Implementar navegação entre imagens (galeria e lightbox).
- Trabalhar com renderização condicional (carrinho vazio vs com itens).
- Organizar a tipagem com TypeScript para props e funções.
- Criar componentes reutilizáveis e bem estruturados.

## 🧠 O que Aprendi
- Gerenciamento de estado com React (useState)
- Tipagem de props com TypeScript
- Manipulação de eventos (click, increment, decrement)
- Componentização e reutilização de código
- Boas práticas de organização de código em projetos React

## 🚀 Melhorias Futuras
- Implementar persistência do carrinho (localStorage)
- Melhorar acessibilidade (aria-labels, navegação por teclado)
- Adicionar animações nas interações
- Implementar testes com Jest e React Testing Library
